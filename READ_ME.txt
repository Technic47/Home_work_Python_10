Commands:
/exit - close program;
/new - creates a new database and ask if you would like to place it as current db;
/show current - shows current db;
/show dir - shows all db`s in db folder;
/select - selects db and place it to current position;
/add - adds a new record to current db;
/show base - shows all records in current db;
/search - start search in current db. First type name of column, then your request;
/export json - exports current db to .json file
/export xml - exports current db to .xml file
/merge - merge selected db to current
/delete - allow you to delete positions in your db
